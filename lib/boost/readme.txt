Boost version 1.5.4

SUSE Linux LIB Build:
----------------------------------------------------------------------------------------------------
- http://www.boost.org/doc/libs/1_54_0/more/getting_started/unix-variants.html#easy-build-and-install
- To solve build errors, may need to install python-devel, libbz2-devel

Windows LIB Build:
----------------------------------------------------------------------------------------------------
- http://www.boost.org/doc/libs/1_54_0/more/getting_started/windows.html#or-simplified-build-from-source
- ./lib32 folder contains some of the prebuilt libs for X86
- ./lib64 are used to extract prebuilt libs for X64
- Prebuilt libs can be downloaded from http://boost.teeks99.com/
